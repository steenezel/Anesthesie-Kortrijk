# Prothese schema
In het EMV is standaard een Prothese PANSAID schema voorzien.
## Toepassingsgebied
Schema is van toepassing op elke:
* schouderprothese (anatomisch + reversed)
* knieprothese (unicondylair + totale + patellofemoraal)
* heupprothese 
* enkelprothese

## Postop
Het is belangrijk om voor elke prothesechirurgie het schema te lanceren. Graag uw aandacht voor:
* voorschrijven van systematisch paracetamol en NSAID (of novalgine bij CI) zowel PO als IV
* doorbraakpijn
    * tradonal IV / po
    * oxynorm instant
* dipidolor / morfine te titreren op PAZA
* dexamethasone 8mg POD1
* fraxiparine 0.3ml ( 0.4ml (>80kg))
* PONV profylaxe
