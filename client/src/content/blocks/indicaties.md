>[!INFO]
> Dit overzicht dient als leidraad voor de blokkeuze bij orthopedische chirurgie in AZ Groeninge. Houd steeds rekening met specifieke voorkeuren van de chirurg.

## Bovenste Lidmaat

| Chirurgie | Block |
| :--- | :--- |
| **Arthroscopie schouder** | ISB <br> *(Needling: geen block; Putzeys AC-resectie: geen block)* |
| **Mobilisatie frozen shoulder** | ISB <br> *(VCDW: catheter; Pierreux: langwerkend single shot)* |
| **Schouderprothese / arthrotomie** | ISB |
| **Proximale humerusfractuur** | ISB ( ! opzoeken n. axillaris uitval) |
| **Humeral shaft fractuur** | ISB ( ! opzoeken n. radialis uitval) |
| **Elleboog / Radiuskap / Prothese** | Supraclaviculair (postop na chir. evaluatie) |
| **Bicepspees herstel** | Supraclaviculair |
| **Fractuur onderarm / pols** | Supraclaviculair |
| **Pinning metacarpaal / falanx** | Polsblock |
| **De Quervain release** | Supraclaviculair <br> *(optie: pols + n. cut. antebrachii lat. / lokaal)* |
| **Carpal tunnel release** | Medianus + Ulnaris + lokale infiltratie |
| **Duimprothese / Polsprothese** | Supraclaviculair |
| **Transpositie n. ulnaris** | AA (Axillair) |
| **Dupuytren** | Supraclaviculair <br> *(JVH: soms polsblock met Naropin bij 1 vinger)* |
| **Springvinger / Extensorpeessutuur** | Polsblock <br> *(JVH: graag motorsparing!)* |

## Onderste Lidmaat

| Chirurgie | Block |
| :--- | :--- |
| **Heupprothese** | Geen block <br> AA of Spinaal (HB Marcaïne + 100 µg Morfine) |
| **Heuparthroscopie** | Geen block *(Fascia Iliaca Compartment te overwegen)* |
| **Knieprothese (Totaal/Uni)** | Femoral Triangle + iPACK |
| **MPFL herstel / Tibia osteotomie** | Femoral Triangle |
| **VKB Reconstructie** | Femoral Triangle |
| **Enkelprothese** | POP-block zonder catheter (20ml Bupivacaïne 0.5%) |
| **Taloniviculaire arthrodese** | POP |
| **Hallux Valgus / Rigidus** | Dewilde/JVDB: POP <br> Michels: geen block |
| **Enkelfractuur** | POP |
| **Mobilisatie knie onder narcose** | CSE of Spinaal + femoraalkatheter (Kinetec?) |

## Algemene Opmerkingen

>[!TIP]
> **Standaard Lokale Anesthetica (Blockroom):**
> * Ropivacaïne 5 mg/ml (Naropin®)
> * Mepivacaïne 20 mg/ml (Scandicaine®)

* **Langwerkende blokken:** Mogelijk met Bupivacaïne 5 mg/ml (Marcaïne®). Isobaar bupivacaïne (20 ml) is voorradig in de zalen van cluster 3 en de VANAS-kast.
* **Fracturen:** Altijd verifiëren of het lidmaat neurovasculair intact is vóór het plaatsen van het block.
* **Mengen van LA:** Bij voorkeur niet mengen. Beperkte winst in onset, maar kortere duur en risico op precipitatie.
* **Longfunctie:** Er zijn geen strikte grenswaarden voor ISB/Supraclaviculair. Bij slechte longfunctie: overweeg Axillair block of selectief Suprascapular/Axillary nerve block.
